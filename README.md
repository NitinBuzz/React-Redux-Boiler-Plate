
# React-Router-Redux-BoilerPlate

My personal React Redux Boiler Plate with Webpack, babel and eslint configured.

## Getting Started

Fork the repo and do yarn install

### Prerequisites

eslint, webpack, babel basic knowledge.

### Installing

yarn init


## Deployment

yarn run dev-server
